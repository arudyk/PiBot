The Goals of PiBot

In an attempt to understand how crucial the AI industry is to technological progression, we will try to create a robot centered on a goal-based agent that uses a knowledge base to infer its partially observable environment. In order to give the project structure, we have created general deadlines for production. The measure of completion will occur at the end of each month. For September, the hardware components for the robot should be installed and a basic website is running that allows users to login and request controller accounts. For October, the website should have a more polished look and the robot will be able to accept movement commands from the web application. For November, we will finalize key features of the web application and begin AI programming. The specifics of this aspect are unclear and as of now the task environment is about the only concrete knowledge we have. And finally December, the robot and the website should be completely functional with limited components of AI potentially working. In the New Year, the completed robot will fill the role of a virtual tour guide that will allow remote connection to control it. All of this while avoiding potentially hazardous situations that would inhibit movement.
